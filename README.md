## WLAN Hosted Network Manager

**Build Status:**

[![CMake on Windows-Latest](https://github.com/BrainB0ne/hostednetwork-gui/actions/workflows/cmake-windows-latest.yml/badge.svg)](https://github.com/BrainB0ne/hostednetwork-gui/actions/workflows/cmake-windows-latest.yml)

**Build Requirements:**

[Windows]

- Microsoft Visual Studio 2019
- Qt SDK 5.15.2 / Qt Creator 18

This project is a GUI application for managing (creating/starting/stopping) 
a WLAN hostednetwork also known as Virtual WiFi Network.

This project is originally developed with Qt 5.11.3

This application executes netsh commands available in Windows 7/8/10
to create a Virtual Wireless Network.

System Requirements: Windows 7/8/10
